# Sprout Homebrew Tap

This is the official Homebrew tap for [Sprout](https://github.com/m44rten1/sprout).

**Sprout** is a CLI tool that manages your Git worktrees so you don't have to. It keeps your project directories clean by tucking worktrees away in `~/.sprout`.
